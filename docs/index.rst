**Welcome to libhal-util API documentation**

Here you will find an organized list of the utilities for the libhal
interfaces, drivers and firmware. This includes soft drivers which are drivers
that emulate specific interface using other drivers, such as an implementation of i2c using output pins (see `hal::bit_bang_i2c`).
